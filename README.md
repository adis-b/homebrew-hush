# homebrew-hush

Homebrew tap for [Hush](https://github.com/adis-b/hush).

```bash
brew install adis-b/hush/hush
```

Or:

```bash
brew tap adis-b/hush
brew install --cask hush
```

Hush is signed ad-hoc, not notarized. First launch needs a right-click → Open
in `Applications`. See the cask caveats for the exact dance.
